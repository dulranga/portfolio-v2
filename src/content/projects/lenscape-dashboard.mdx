# Lenscape Dashboard

Welcome to the technical breakdown of **Lenscape Dashboard**. This project was born out of a need for photographers to have a real-time, high-performance view of their gear and schedules.

## Core Features
1. **Real-time Analytics**: Built using React Query for efficient state management.
2. **Dynamic Scheduling**: A custom-built calendar interface.
3. **High Contrast UI**: Designed with a focus on legibility in low-light environments.
